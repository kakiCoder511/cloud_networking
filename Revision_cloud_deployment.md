## What is Cloud Computing?
Computing resources and services accessed over the internet, on demand.

## Why is it popular?
- Enchanced collaboration
- Easy/ fast deployment and prototype asessment
- No phyical costs/investment up front
    - On demand pricing
- Scalability
-Accessibility
    -Only need an internet connection and valid cloud credentials
-Responsibility often moves to cloud provider
- Faster global deployment
- Innovation!
    - Easier to experiment and try new te

## Cloud deployment model

**Public Cloud**

- mutli-tenacny( e.g. AWS, Azure, GCP)

**Private Cloud**
- single- Tenacny (e.g. gov-cloud)

**Hybrid Cloud**

- Maxture of public and private

**Multi Cloud**

- Mixture of multiple public cloud providers

## Service types

**SaaS --> Software as a Service**

Full application hosted and running on the cloud.

You simply provide the data.
e.g. Office 365, Email

**PaaS --> Platform as a Service**

Cloud provider gives an environment to use for development. You provide the code and the data.

**IaaS --> Infraststructure as Servive**

Hardware only, you must configure everything else. Most control and most complex.

**FaaS --> Functions as a Service**

On the spot function execution.
